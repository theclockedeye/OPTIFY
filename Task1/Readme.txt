# Unit Calculator

## Overview
The Unit Calculator is a simple web application designed to facilitate unit conversions. It allows users to easily convert between various units of measurement, making it a handy tool for students, professionals, and anyone who needs to perform quick conversions.

## Features
- **Multiple Unit Types**: Supports a variety of unit types including length, weight, temperature, and more.
- **User-Friendly Interface**: Designed for easy navigation with a clean and intuitive layout.
- **Real-Time Conversion**: Instant conversion results as users input values.
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.

## Technologies Used
- **HTML**: For structuring the content.
- **CSS**: For styling the application and enhancing the user experience.
- **JavaScript**: For implementing the conversion logic and real-time calculations.

## Installation
To run the Unit Calculator locally, follow these steps:

1. Clone the repository:
   git clone https://github.com/theclockedeye/OPTIFY.git

2. Navigate to the project directory:
  cd OPTIFY/task1
#Usage
1. Select the unit type you wish to convert from the dropdown menu.
2. Enter the value you want to convert.
3. Choose the target unit to convert to.
4. Click the "Convert" button to see the result.
